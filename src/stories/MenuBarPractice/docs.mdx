# MenuBar - 2Depth

## 키보드 컨트롤

### 상위 메뉴 항목에 초점이 있는 경우

### 1. `Arrow Left`

- `Arrow Left` 키를 통해 이전 상위 메뉴 항목으로 초점을 이동시킵니다.
  <br />
  첫번째 상위 메뉴 항목에서 `Arrow Left` 키를 누르면 마지막 상위 메뉴 항목으로 초점을
  이동시켜 상위 메뉴 항목 내에서 초점이 순환되도록 합니다.

### 2. `Arrow Right`

- `Arrow Right` 키를 통해 다음 상위 메뉴 항목으로 초점을 이동시킵니다.
  <br />
  마지막 상위 메뉴 항목에서 `Arrow Right` 키를 누르면 첫번째 상위 메뉴 항목으로 초점을
  이동시켜 상위 메뉴 항목 내에서 초점이 순환되도록 합니다.

### 3. `Home`

- `Home` 키를 통해 첫번째 상위 메뉴 항목으로 초점을 이동시킵니다.

### 4. `End`

- `End` 키를 통해 마지막 상위 메뉴 항목으로 초점을 이동시킵니다.

### 5. `Arrow Down`

- 드롭다운 메뉴가 있는 상위 메뉴 항목의 경우 `Arrow Down` 키를 통해 드롭다운 메뉴를 펼칩니다.
  <br />
  동시에 `Arrow Down` 키를 통해서 드롭다운 메뉴가 열린 경우에는 첫번째 하위 메뉴
  항목으로 초점이 이동됩니다.

### 6. `Arrow Up`

- 드롭다운 메뉴가 있는 상위 메뉴 항목의 경우 `Arrow Up` 키를 통해 드롭다운 메뉴를 펼칩니다.
  <br />
  동시에 `Arrow Up` 키를 통해서 드롭다운 메뉴가 열린 경우에는 마지막 하위 메뉴 항목으로
  초점이 이동됩니다.

### 7. `Esc`

- 드롭다운 메뉴가 열려 있는 상위 메뉴 항목에서 `Esc` 키를 누르면 드롭다운 메뉴를 닫습니다. 이때 초점은 해당 상위 메뉴 항목으로 이동됩니다.
  <br />
  `Esc` 키를 통해 드롭다운 메뉴를 닫는 경우 사용자의 의지에 의해 닫힌 것으로 생각하여
  다시 드롭다운 메뉴를 펼치는 키보드 동작을 하지 않는 한 드롭다운 메뉴가 자동으로
  열리지 않도록 합니다. 예를 들어 `Arrow Left`와 `Arrow Right` 키를 통해 상위 메뉴
  항목간 초점을 이동시켜도 드롭다운 메뉴가 자동으로 열리지 않습니다.

### 8. `Tab`

- 드롭다운 메뉴가 열려 있는 상위 메뉴 항목에서 `Tab` 키를 누르면 메뉴바 외부 초점이 가능한 요소로 초점이 이동되고 드롭다운 메뉴를 닫습니다.
  <br />
  `Tab` 키를 통해 드롭다운 메뉴가 닫히고 나면 `Esc` 키와 동일하게 다시 드롭다운 메뉴를
  펼치는 키보드 동작을 하지 않는 한 드롭다운 메뉴가 자동으로 열리지 않도록 합니다.

### 9. `Enter` 또는 `Space`

- 드롭다운 메뉴가 없는 상위 메뉴 항목의 경우 `Enter` 또는 `Space` 키를 통해 해당 상위 메뉴 항목을 선택합니다.
  <br />
  드롭다운 메뉴가 있는 상위 메뉴 항목의 경우 `Enter` 또는 `Space` 키를 통해 드롭다운
  메뉴를 열고 첫번째 하위 메뉴 항목으로 초점을 이동시킵니다. 이미 드롭다운 메뉴가
  열려있는 경우에도 동일하게 첫번째 하위 메뉴 항목으로 초점을 이동시킵니다.

<br />

  <br />
  `Esc` 키를 통해 드롭다운 메뉴가 닫히고 나면 상위 메뉴 항목에서의 `Esc`키 동작과
  동일하게 다시 드롭다운 메뉴를 펼치는 키보드 동작을 하지 않는 한 드롭다운 메뉴가
  자동으로 열리지 않도록 합니다.

### 8. `Tab`

- 하위 메뉴 항목에서 `Tab` 키를 누르면 메뉴바 외부 초점이 가능한 요소로 초점이 이동되고 드롭다운 메뉴를 닫습니다.

## aria
- aria-selected: 현재 선택된 menuitem
- aria-current: 선택된 leaf menuitem 에 'page'
- aria-expanded: submenu 가 있는 상위 menuitem 에
- aria-haspopup: submenu 가 있는 상위 menuitem 에
- aria-label: nav 와 role='menubar' 에
