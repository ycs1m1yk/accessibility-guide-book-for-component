# SelectMenuMultiple

## 키보드 컨트롤

### 옵션 목록 팝업이 닫혀있는 경우

### 1. `Arrow Down` 또는 `Arrow Up` 또는 `Space` 또는 `Enter`

- 4개의 키 동작 모두 옵션 목록 팝업을 열고 마지막으로 활성화 되었던 옵션이 활성화 되도록 합니다.

<br />

### 옵션 목록 팝업이 열려있는 경우

### 1. `Arrow Up`

- `Arrow Up` 키를 통해 이전 옵션을 활성화 시킵니다.

### 2. `Arrow Down`

- `Arrow Down` 키를 통해 다음 옵션을 활성화 시킵니다.

### 3. `Home` (옵션)

- `Home` 키를 통해 첫번째 옵션을 활성화 시킵니다.

### 4. `End` (옵션)

- `End` 키를 통해 마지막 옵션을 활성화 시킵니다.

### 5. `Page Up` (옵션)

- `Page Up` 키를 통해 현재 활성화된 옵션으로부터 10번째 이전 옵션을 활성화 시킵니다.
  <br />
  남아있는 이전 옵션의 개수가 10개 미만인 경우, 첫번째 옵션을 활성화 시킵니다.

### 6. `Page Down` (옵션)

- `Page Down` 키를 통해 현재 활성화된 옵션으로부터 10번쩨 다음 옵션을 활성화 시킵니다.
  <br />
  남아있는 다음 옵션의 개수가 10개 미만인 경우, 마지막 옵션을 활성화 시킵니다.

### 7. `Space` 또는 `Enter`

- `Space` 또는 `Enter` 키를 통해 현재 활성화되어 있는 옵션 값을 선택하고 옵션 목록 팝업을 닫습니다.

### 8. `Esc`

- `Esc` 키를 통해 옵션 목록 팝업을 닫습니다.

### 9. 활성화된 옵션이 시각적으로 보여지게 해야 합니다.

- 옵션 목록 팝업의 스크롤에 의해 활성화된 옵션이 시각적으로 보이지 않는 경우, 활성화된 옵션이 항상 시각적으로 보여지도록 팝업의 스크롤을 조정해야 합니다.



















## 요구사항

- 다중 선택이 가능해야합니다.
- Select의 Placeholder는 `${첫번째 선택된 Option} 외 N개` 로 표현합니다.
- Select Button 을 클릭하면, 옵션 listbox 가 떠야합니다.
- 키보드 대응
  - 옵션 listbox 가 열려있는 경우
    - `위 화살표` 를 누르면, 활성화 된 영역이 위로 하나씩 이동한다. 끝까지 올라가면 활성화 영역은 멈춘다.
    - `아래 화살표` 를 누르면, 활성화 된 영역이 아래로 하나씩 이동한다. 끝까지 내려가면 활성화 영역은 멈춘다.
    - `스페이스`, `엔터` 를 누르면, 옵션이 선택(선택안됨) 된다.
    - `ESC` 를 누르면, listbox 가 닫힌다.
  - 옵션 listbox 가 닫혀있는 경우
    - `위 화살표`, `아래 화살표`, `스페이스`, `엔터` 를 누르면 listbox 가 열린다.
