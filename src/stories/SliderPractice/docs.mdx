# Slider

## 키보드 컨트롤

### 초점이 핸들 요소에 있는 경우

### 1. `ArrowLeft` 또는 `ArrowDown`

- `ArrowLeft` 키 또는 `ArrowDown` 키를 통해 슬라이더 현재 값을 감소시킵니다. 슬라이더의 최소값 범위에 도달할 때까지 값을 감소시킬 수 있습니다.

### 2. `ArrowRight` 또는 `ArrowUp`

- `ArrowRight` 키 또는 `ArrowUp` 키를 통해 슬라이더 현재 값을 감소시킵니다. 슬라이더의 최소값 범위에 도달할 때까지 값을 감소시킬 수 있습니다.

### 3. `Home`

- `Home` 키를 통해 핸들이 슬라이더의 최소값으로 이동합니다. 사용자는 슬라이더에서 허용되는 가장 작은 값을 선택할 수 있습니다.

### 4. `End`

- `End` 키를 통해 핸들이 슬라이더의 최대값으로 이동합니다. 사용자는 슬라이더에서 허용되는 가장 큰 값을 선택할 수 있습니다.

### 5. `Page Up` (옵션)

- `Page Up` 키를 통해 현재 값이 전체 범위의 10% 만큼 증가합니다.

### 6. `Page Down` (옵션)

- `Page Down` 키를 통해 현재 값이 전체 범위의 10% 만큼 감소합니다.
